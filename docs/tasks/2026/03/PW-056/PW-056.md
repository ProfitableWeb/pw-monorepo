## 🎯 Постановка задачи

### Описание

Добавить модуль генерации и редактирования Table of Contents (оглавления) для статей. ToC генерируется автоматически из
заголовков H2–H4 контента, с возможностью включить/исключить отдельные пункты.

### Цель

Автоматическое оглавление для навигации по длинным статьям. Данные сохраняются в JSONB-поле `toc` модели Article.

### Критерии приемки

- [x] Backend: JSONB-поле `toc` в модели Article + миграция
- [x] Backend: поле `toc` в Create/Update/Response схемах
- [x] Admin: иконка ToC (List) в EditorToolbar рядом с Format и Settings
- [x] Admin: модальное окно с парсингом заголовков H2–H4 из HTML
- [x] Admin: чекбоксы для включения/исключения пунктов, иерархический отступ
- [x] Admin: бэйдж с количеством включённых пунктов на иконке
