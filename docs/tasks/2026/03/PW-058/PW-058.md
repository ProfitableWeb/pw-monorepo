# PW-058: Article header flush alignment with layout grid

## Информация о задаче

- **ID**: PW-058
- **Статус**: DONE
- **Завершено**: 2026-03-20
- **Создано**: 2026-03-20
- **Приоритет**: Normal
- **Компонент**: Web
- **Теги**: #layout #css #alignment #article-page #responsive

## Постановка задачи

### Описание

ArticleHeader (заголовок, подзаголовок, мета) рендерится **над** CSS Grid лейаутом (ArticleLayout) и выравнивается
независимо (`max-width: 70ch` + `margin: 0 auto`). Из-за этого левый край шапки не совпадает с левым краем контентной
колонки grid — нет выравнивания «заподлицо».

### Решение

Переместить ArticleHeader **внутрь** ArticleLayout как участника grid. Header получает `grid-column: 1 / -1` и занимает
всю ширину grid во всех layout-вариантах.

### Критерии приемки

- [x] ArticleHeader рендерится внутри ArticleLayout grid
- [x] Header спанится `grid-column: 1 / -1` во всех layout-типах
- [x] Левый край header совпадает с левым краем grid на всех брейкпоинтах
- [x] Визуально корректно на: mobile (<640px), tablet (640-900px), tablet (900-1200px), desktop (1200-1400px), wide
      (1400px+)
- [x] Работает для всех 4 layout-типов: three-column, two-column, one-column, full-width

## План реализации

### TSX-компоненты

1. `ArticleLayout.tsx` — добавить проп `header?: ReactNode`, прокинуть в каждый layout
2. `ArticleLayoutThreeColumn.tsx` — принять `header`, рендерить `<div className="__header">{header}</div>`
3. `ArticleLayoutTwoColumn.tsx` — аналогично
4. `ArticleLayoutOneColumn.tsx` — аналогично
5. `ArticleLayoutFullWidth.tsx` — аналогично
6. `ArticlePage.tsx` — переместить `<ArticleHeader>` из `<main>` в `<ArticleLayout header={...}>`

### SCSS-стили

7. В каждом layout SCSS — добавить `&__header { grid-column: 1 / -1; }`
8. `ArticleHeader.scss` — убрать `max-width: 70ch` + `margin: 0 auto` (выравнивание теперь от grid)

## Заметки

### Логика header по брейкпоинтам

**Three-column:**

- < 900px (одна колонка): на всю ширину
- 900-1400px (две колонки — sidebar виден, ToC скрыт): `grid-column: 1 / -1` — обе колонки
- 1400px+ (три колонки): `grid-column: 2` — центральная колонка

**Two-column:**

- < 900px (одна колонка): на всю ширину
- 900px+ (две колонки): `grid-column: 1 / -1` — обе колонки

**One-column / Full-width:** header на всю ширину (нет grid)

### Дополнительный фикс

Убран `margin-top` у первого элемента контента (`> :first-child { margin-top: 0 }`) — контент плотно прилегает к header.

### Контекст

Проблема стала видна после наполнения БД реальным контентом и переключения ArticlePage со стаба на полноценный рендеринг
(ArticleLayout + ArticleHeader).
