# PW-050: Стилизация sitemap.xml — XSL-шаблон с dark-темой и логотипом PW

**Статус**: DOING **Создано**: 2026-03-17 **Приоритет**: Normal **Компонент**: ⚛️ Web

**Зависимости**: PW-049 (sitemap.xml генерация)

## Проблема

Браузер отображает sitemap.xml как сырой XML без форматирования. Для SEO-инструментов это не важно, но при открытии
человеком (разработчик, SEO-специалист) хочется видеть оформленную страницу.

## Цель

Стилизованный sitemap.xml через XSL-трансформацию: dark-тема, логотип PW, таблица URL с приоритетами и датами.
Аналогично для feed.xml (RSS/Atom).
