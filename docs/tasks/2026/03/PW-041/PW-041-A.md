## 1. Модель MediaFile

Файл: `apps/api/src/models/media_file.py`

```python
class MediaFile(Base, TimestampMixin):
    __tablename__ = "media_files"

    id: Mapped[UUID] = mapped_column(primary_key=True, default=uuid4)
    filename: Mapped[str]                         # оригинальное имя при загрузке
    storage_key: Mapped[str] = mapped_column(unique=True)  # "media/{uuid}.webp"
    mime_type: Mapped[str]                        # image/webp, audio/mpeg, application/pdf
    file_type: Mapped[str]                        # "image" | "video" | "audio" | "document"
    size: Mapped[int]                             # байты (оригинал)

    # Изображения
    width: Mapped[int | None] = mapped_column(default=None)
    height: Mapped[int | None] = mapped_column(default=None)

    # SEO
    slug: Mapped[str] = mapped_column(unique=True)  # SEO-friendly, уникальный, редактируемый
    alt: Mapped[str | None] = mapped_column(default=None)    # alt text, ≤125 символов
    caption: Mapped[str | None] = mapped_column(default=None)  # подпись, ≤250 символов

    # EXIF (только изображения)
    exif_data: Mapped[dict | None] = mapped_column(JSONB, default=None)

    # Организация (свободный ввод)
    purposes: Mapped[list[str]] = mapped_column(JSONB, default=list)

    # Ресайзы (генерируются при загрузке)
    resizes: Mapped[list[dict] | None] = mapped_column(JSONB, default=None)

    # Связи
    uploaded_by_id: Mapped[UUID] = mapped_column(ForeignKey("users.id"))
    uploaded_by: Mapped["User"] = relationship(back_populates="media_files")
    articles: Mapped[list["Article"]] = relationship(
        secondary="article_media", back_populates="media_files"
    )
```

## 4. Media service

Файл: `apps/api/src/services/media.py`

### 4.1. Upload pipeline

```
upload(file: UploadFile, user_id: UUID) → MediaFile:
    1. Валидация MIME + размер (≤20 MB для изображений)
    2. Генерация UUID, slug из filename (python-slugify, transliterate)
    3. Определение file_type по MIME:
       image/* → "image"
       video/* → "video"
       audio/* → "audio"
       * → "document"
    4. Если image:
       a. Pillow.open() → получить width, height
       b. Извлечь EXIF (Pillow ExifTags)
       c. Конвертировать в WebP (quality 85)
       d. Resize pipeline → 4 варианта (если оригинал больше)
       e. Сохранить оригинал + ресайзы через StorageBackend
    5. Если не image:
       a. Сохранить файл как есть через StorageBackend
    6. Создать запись MediaFile в БД
    7. Вернуть MediaFile
```

### 4.2. Resize pipeline

По аналогии с `avatar.py`, но 4 размера:

| Суффикс  | Размер           | Метод                         |
| -------- | ---------------- | ----------------------------- |
| (без)    | оригинал         | WebP conversion               |
| `_thumb` | 150×150          | crop center (как в avatar.py) |
| `_sm`    | 400px по ширине  | proportional resize           |
| `_md`    | 800px по ширине  | proportional resize           |
| `_lg`    | 1200px по ширине | proportional resize           |

Если оригинал меньше целевого размера — ресайз не создаётся. Все ресайзы — WebP, quality 85.

Storage paths:

```
media/{uuid}.webp
media/{uuid}_thumb.webp
media/{uuid}_sm.webp
media/{uuid}_md.webp
media/{uuid}_lg.webp
```

### 4.3. EXIF extraction

Из Pillow ExifTags, маппинг в удобный dict:

```python
{
    "date_taken": "2025-06-15T14:30:00",
    "camera": "Canon EOS R5",
    "lens": "RF 24-70mm F2.8",
    "iso": "400",
    "aperture": "f/2.8",
    "shutter_speed": "1/250",
    "focal_length": "50mm"
}
```
