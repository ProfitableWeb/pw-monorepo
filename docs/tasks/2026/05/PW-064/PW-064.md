## Постановка задачи

### Описание

В проекте нет механизма подключения к LLM-провайдерам. Необходимо реализовать полноценное управление AI-провайдерами:
хранение ключей, конфигурация моделей, поддержка разных типов API.

Поддерживаемые типы API:

- **OpenAI-compatible** — OpenAI, vLLM, Ollama, Groq, Together, OpenRouter, любой OpenAI-совместимый сервер
- **Anthropic** — Claude API напрямую

### Цель

Администратор может через UI настроить подключение к любому LLM-провайдеру, указав API key, хост, модель и тип API.
Ключи хранятся зашифрованно. Один провайдер помечается как активный по умолчанию.

### Критерии приемки

- [ ] Модель `AiProvider` в БД (name, api_type, api_key encrypted, base_url, model_name, is_default, is_active,
      max_context_tokens, created_at, updated_at)
- [ ] Миграция Alembic создана и применяется
- [ ] API CRUD для провайдеров (admin-only): list, create, update, delete, set_default, test_connection
- [ ] Шифрование API-ключей (Fernet, ключ из переменной окружения)
- [ ] Эндпоинт test_connection — проверка подключения через простой запрос к LLM
- [ ] API-ключ маскируется при отдаче (показываются первые 4 и последние 4 символа)
- [ ] Админка: раздел «Провайдеры» в AI Center с CRUD-интерфейсом
- [ ] Валидация: обязательные поля, формат URL, минимальная длина API key
- [ ] Нельзя удалить единственного активного провайдера

### Зависимости

- [ ] `cryptography` (Fernet) — уже в зависимостях проекта
- [ ] Переменная окружения `AI_ENCRYPTION_KEY` для шифрования ключей

### Техническое решение

#### 1. Модель БД

```python
# apps/api/src/models/ai_provider.py
class ApiType(str, enum.Enum):
    OPENAI_COMPATIBLE = "openai_compatible"
    ANTHROPIC = "anthropic"

class AiProvider(Base):
    __tablename__ = "ai_providers"

    id: Mapped[uuid.UUID] = mapped_column(primary_key=True, default=uuid.uuid4)
    name: Mapped[str] = mapped_column(String(100))
    api_type: Mapped[ApiType]
    api_key_encrypted: Mapped[str] = mapped_column(Text)
    base_url: Mapped[str | None] = mapped_column(String(500))
    model_name: Mapped[str] = mapped_column(String(200))
    is_default: Mapped[bool] = mapped_column(default=False)
    is_active: Mapped[bool] = mapped_column(default=True)
    max_context_tokens: Mapped[int | None] = mapped_column()
    # TimestampMixin: created_at, updated_at
```
