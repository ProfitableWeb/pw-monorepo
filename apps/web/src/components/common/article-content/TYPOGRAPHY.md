# Типографика Article Content

## Основные принципы

1. **Читаемость превыше всего** - оптимальная ширина строки 65-75 символов (75ch)
2. **Заметная разница между разрешениями** - размеры растут пропорционально экрану
3. **Умеренный line-height** - 1.6-1.7 для основного текста (не слишком разрежено)
4. **Lead paragraph выделяется** - первый параграф крупнее для привлечения внимания

## Адаптивные размеры шрифта

### Основной текст (параграфы)

| Разрешение               | font-size | line-height | Когда используется |
| ------------------------ | --------- | ----------- | ------------------ |
| Mobile (<640px)          | 14px      | 1.5         | Смартфоны          |
| Tablet Small (640-900px) | 15px      | 1.55        | Планшеты portrait  |
| Tablet (900-1200px)      | 16px      | 1.6         | Планшеты landscape |
| Desktop (1200-1600px)    | 17px      | 1.6         | Ноутбуки, мониторы |
| Wide Desktop (>1600px)   | 18px      | 1.6         | Большие мониторы   |

### Lead Paragraph (вводный параграф)

**Что это:** Первый параграф статьи, который резюмирует основную информацию и привлекает внимание читателя.

**Как применяется:**

- Автоматически к первому `<p>` в статье
- Или через класс `.lead` для явного указания

| Разрешение               | font-size | line-height | Разница с основным |
| ------------------------ | --------- | ----------- | ------------------ |
| Mobile (<640px)          | 16px      | 1.45        | +2px               |
| Tablet Small (640-900px) | 17px      | 1.5         | +2px               |
| Tablet (900-1200px)      | 18px      | 1.5         | +2px               |
| Desktop (1200-1600px)    | 19px      | 1.5         | +2px               |
| Wide Desktop (>1600px)   | 20px      | 1.5         | +2px               |

### Заголовки

| Элемент | Размер | line-height | Примечания                    |
| ------- | ------ | ----------- | ----------------------------- |
| h1      | 2.25em | 1.2         | Только для главного заголовка |
| h2      | 1.75em | 1.25        | С нижней границей             |
| h3      | 1.5em  | 1.3         | Подзаголовки секций           |
| h4      | 1.25em | 1.25        | Мелкие разделы                |
| h5      | 1.1em  | 1.25        | Редко используется            |
| h6      | 1em    | 1.25        | Редко используется            |

## Использование

### HTML структура

```html
<article class="article-content">
  <!-- Lead paragraph (автоматически крупнее) -->
  <p>
    Это вводный параграф, который резюмирует статью и привлекает внимание. Он автоматически будет крупнее остальных
    параграфов.
  </p>

  <!-- Обычные параграфы -->
  <p>Это обычный параграф с базовым размером шрифта.</p>
  <p>И еще один параграф.</p>

  <!-- Явное указание lead paragraph -->
  <p class="lead">
    Можно также явно указать lead paragraph через класс, если первый параграф по какой-то причине не должен быть lead.
  </p>
</article>
```

### Работа с разными разрешениями

Все размеры адаптируются автоматически через медиа-запросы. Вам не нужно ничего указывать дополнительно.

**Пример на разных устройствах:**

```
iPhone SE (375px):
- Основной текст: 14px, line-height: 1.5
- Lead: 16px, line-height: 1.45
- h2: 24px (1.75em от 14px)

iPad (1024px):
- Основной текст: 16px, line-height: 1.6
- Lead: 18px, line-height: 1.5
- h2: 28px (1.75em от 16px)

Desktop (1440px):
- Основной текст: 17px, line-height: 1.6
- Lead: 19px, line-height: 1.5
- h2: 30px (1.75em от 17px)

Wide Monitor (1920px):
- Основной текст: 18px, line-height: 1.6
- Lead: 20px, line-height: 1.5
- h2: 31px (1.75em от 18px)
```

## Настройка

### Где менять размеры шрифта?

Все адаптивные размеры в одном месте: → `article-content/_responsive-typography.scss`

### Пример изменения

**Хочу увеличить lead paragraph на Mobile:**

```scss
// В _responsive-typography.scss
.article-content {
  font-size: 15px;

  p:first-of-type,
  .lead {
    font-size: 18px; // Было 17px, стало 18px
  }
}
```

**Хочу изменить line-height для основного текста:**

```scss
@media (min-width: 900px) {
  .article-content {
    font-size: 17px;
    line-height: 1.7; // Было 1.65, стало 1.7
  }
}
```

## Особенности

### Оптимальная ширина строки

Все параграфы и списки имеют `max-width: 75ch` для оптимальной читаемости (65-75 символов в строке).

### Line-height заголовков

Заголовки имеют более компактный line-height (1.2-1.3), чем основной текст, для визуальной плотности.

### Цитаты

Цитаты немного крупнее основного текста (`font-size: 1.05em`) и имеют курсивное начертание.

## Советы

✅ **DO:**

- Используйте `p:first-of-type` для автоматического lead paragraph
- Добавляйте класс `.lead` только когда нужно явно указать lead
- Тестируйте читаемость на реальных устройствах
- Проверяйте, что разница между разрешениями заметна

❌ **DON'T:**

- Не делайте line-height слишком большим (>1.8 для основного текста)
- Не делайте слишком маленькую разницу между разрешениями
- Не забывайте про max-width для читаемости
- Не игнорируйте lead paragraph - он важен для UX

## Тестирование

Проверьте типографику на:

1. Разных размерах экрана (375px, 768px, 1024px, 1440px, 1920px)
2. Длинных и коротких параграфах
3. Разных языках (кириллица, латиница)
4. Светлой и темной теме
